About

Data Center is an open online platform that provides easy access to datasets of various public information at the national and local level in Serbia. It was developed to help journalists, activists, researchers and public officials in search for valuable data to power their stories, campaigns, analyses or decision making. With the use of latest technologies, Data Center simplifies, integrates and prepares complex, disparate and disorganized datasets for easy and meaningful use by all interested stakeholders. By making good use of public data, Data Center and its users push for governments to offer useful and good quality data and provide the public what they need to participate in decision making as active and informed citizens.

Ways to Use the Site

Advanced search

  • You can start exploring right from the homepage: click on the pre-made search right below the video, or click on “show me another” to get a new pre-made search
  • On the Search page, you can visualize all our data and filter it by any variable (like geographic location, amount, or anything else)
  • The visualization are automatically updated, to give you an easier overview of what the data is saying
  • We are working on custom advanced searches, where you will be able to define operators like “less than”, “is true”, “is empty” etc to create complex filtering directly on the website
  • You can also download the data in different open formats for your own analyses
  • Don’t know what an API is? The School of Data explains APIs for non-programmers (but you should probably get a developer to help you - or ask us for help!)
  • On the API page you can learn about all the parameters you can use for your API calls, and create your custom API
  • Run the custom API directly on the Data Centar website to see what results it provides you, then tweak to your delight
  • Once you’re happy, take the API and plug it into your website, or use it to download raw data on your computer
  • Our entire Search page is powered by our API - so you can create advanced searches, and copy the API link of the search you made
  • At the bottom of each page, you will find a form where you can choose between four datasets we plan to open.
  • Take your pick, and if you want, leave us a comment on how you would use that data. Your feedback is crucial!
  • If you’re not interested in any of the proposed datasets, get in touch and propose what you would like us to liberate next
